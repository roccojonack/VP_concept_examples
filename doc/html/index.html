This is the main page of the documentation for the SSRA project. A number of links are used to reference the documentation of different components used in the projet. Each of those component has in turn documentation like doxygen based source docuementation, lists of build errors/warnings, results of regression runs and statistics on code. Not all components have the same characteristics and test output, hence some of the information is different per component.

There is also a section for the general documentation of the SSRA project, which summerizes a lot of the information from the different tests.
